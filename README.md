### Answer 1
```html
<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8"/>
  <meta name="viewport" content="width=device-width,initial-scale=1"/>
  <title>Solution Q1</title>
</head>
<body>
  <!-- Paragraph with Hello World text -->
  <p>Hello World! This is my first web page.</p>
</body>
</html>
```
